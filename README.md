# babycareai-spring-boot

아래 정보를 기준으로 개발중입니다.

### 기능
- 이미지 업로드
- 이미지 분석 요청
- 가이드 요청
- 추가 정보 입력 (병변 위치, 발열 여부, 가려움, 염증 등) (후순위)
- 인증/인가 (후순위)
- 이미지 메타데이터 저장 (사용자ID, 업로드된 이미지의 S3 URL, 이미지 업로드 시간) (선택)
